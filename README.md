## 🚀 Quick start

### Instalação

```bash
$ git clone https://github.com/jeferson-sb/aircnc.git && cd aircnc
$ cd backend && yarn
```

```bash
$ cd .. & cd frontend
$ yarn
```

### Utilização

```
Renomeie os arquivos .env.example para .env
```

```bash
$ cd backend
$ yarn dev
```

Abra uma outra aba no terminal, abra pasta raíz do projeto e rode

```bash
$ cd frontend
$ yarn start
```
