### Demo : [https://aircnc-web.herokuapp.com/](https://aircnc-web.herokuapp.com/)

## Quick Start (Web)

```
$ git clone https://github.com/jeferson-sb/aircnc.git && cd aircnc
```

```
$ cd backend && yarn
$ yarn dev
```

Abra uma outra aba no terminal, abra pasta raíz do projeto e rode
```
$ cd frontend && yarn
$ yarn start
```

## Install on mobile

```
$ cd mobile
$ yarn && yarn start
```
